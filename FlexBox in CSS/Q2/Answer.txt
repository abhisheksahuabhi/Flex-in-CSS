
1. Justify-Content

The justify-content property aligns the flexible container's items when the items do not use all 
available space on the main-axis (horizontally).
The justify-content property can also be used on a grid container to align grid items in the inline direction.

2. Align-items
The justify-items property is set on the grid container to give child elements (grid items) 
alignment in the inline direction

3. Gap

The gap property defines the size of the gap between the rows and between the columns in flexbox, grid or multi-column layout. 
It is a shorthand for the following properties:
    a. row-gap
    b. column-gap

4. Flex-direction
The flex-direction property specifies the direction of the flexible items.

a. row	Default value :: The flexible items are displayed horizontally, as a row	
b. row-reverse :: Same as row, but in reverse order	
c. column :: The flexible items are displayed vertically, as a column	
d. column-reverse :: Same as column, but in reverse order

5. Flex-wrap

The flex-wrap property specifies whether the flexible items should wrap or not.
a. nowrap	:: Default value. Specifies that the flexible items will not wrap	
b. wrap	:: Specifies that the flexible items will wrap if necessary	
c. wrap-reverse	:: Specifies that the flexible items will wrap, if necessary, in reverse order
